СОЗДАНИЕ СИСТЕМЫ АКТИВНЫХ ЭЛЕМЕНТОВ И НАВИГАЦИЯ ПО НИМ:

ОБЩАЯ КОНЦЕПЦИЯ:

Каждая страница имеет свой массив индексов активных элементов.
Предполагается всегда отводить нулевой индекс под свойства экрана



ЗАДАЧИ:

1) Написать функционал и данные для базового класса ActiveElement, и подумать как реализовать наследование
2) Унаследовать ActiveElements (и остальные классы активных элементов от класса Page)
3) Написать инструмент для вставки текста с расширенными возможностями редактирования цвета
4) Написать инструмент для использования текущего положения курсора
5) Метод update в каждом классе page будет одинаков, следовательно и контент везде будет повторяться. Надо
реализовать разделение контента для каждого отдельного класса
6) В конце концов отказаться от ActiveElement класса как основного для создания элементов и унаследовать 
все классы активных элементов от одного базового класса, который будет определять общие для всех элементов параметры и данные
7) При отрисовке активных элементов в конце вставлять очистку цветовых параметров, дабы избежать окрашивания всего текста в консоли
